# Contributing to Agent Trust Suite

Thanks for your interest in contributing!

## Where to Contribute

Agent Trust Suite is split across multiple repositories. Please file issues and PRs in the relevant repo:

| Repository | What it does |
|---|---|
| [agentcontract](https://github.com/wharfe/agentcontract) | Contract definition & validation |
| [agent-trust-telemetry](https://github.com/wharfe/agent-trust-telemetry) | Runtime violation detection |
| [trustbundle](https://github.com/wharfe/trustbundle) | Evidence trail packaging |
| [agentbond](https://github.com/wharfe/agentbond) | Authorization & governance |

For **cross-repo** topics (architecture, new layer proposals, demo scenarios), use this repository's [Issues](https://github.com/wharfe/agent-trust-suite/issues).

## Getting Started

1. Fork the relevant repository
2. Create a branch (`git checkout -b my-change`)
3. Make your changes
4. Run tests (each repo has its own test setup — see its README)
5. Commit using [Conventional Commits](https://www.conventionalcommits.org/) format
6. Open a pull request

## Conventions

- **Commit messages:** Conventional Commits (`feat:`, `fix:`, `docs:`, etc.)
- **Code comments:** English
- **Tests:** Required for new functionality

## Questions?

Open an issue in this repo or in the relevant component repo. We're happy to help.
